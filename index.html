About EchoTube

What is EchoTube?

EchoTube is a experiment in the works. An algorithm free platform, where you can search the massive video collections of Youtube, pick your favorite videos, bookmark them in one simple spot, and come back to view them later.

How Does EchoTube work?

  • Search with the official Youtube Data API created by Google.
  • API Keys stay in the server where EchoTube runs. Zero chance of leaking your key to the public. (This does mean you will need a FREE api key)
  • Bookmarks are stored in your browser's localStorage. They don't sync across devices, but they also don't require an account.

What Could be next?

  • Vault: Archive Creative Commons-licensed videos for offline viewing.
  • Channel Backup: Connect your YouTube account to back up your own uploads.
  • Sync: Optional account system so bookmarks follow you across devices.

Built using HTML, CSS, Bootstrap, Vanilla JavaScript, and Node.js + Express.
